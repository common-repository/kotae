== Description ==

Turn your existing posts into Help / Knowledge base content!

Kotae is a knowledge base plugin that lets you tag your existing posts and organize them into a knowlege base.
Unlike most other knowledge base / FAQ plugins you don't need to create custom posts. You can organize your existing content in any way you wish without rewriting stuff.

See what Kotae does:

###Features###

- Lets you tag existing posts and pages as knowledge base articles.
- Provides a shortcode for publishing the main knowledge base entry page.
- Lets you organize the content in up to 10 columns.
- Lets you control how many article links are shown for each topic.
- Provides thumbs up / thumbs down or five-star rating widget so you can collect feedback about how useful your articles are.
- Lets you manage and reorder the knowledge base categories (tags)

== Installation ==

1. Unzip the contents and upload the entire `kotae` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Kotae" in your menu and manage the plugin
